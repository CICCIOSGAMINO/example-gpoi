# Github Pages

https://docs.github.com/en/pages/getting-started-with-github-pages

Getting start Github
https://docs.github.com/en/pages/getting-started-with-github-pages/creating-a-github-pages-site

Github pages è un servizio di hosting statico direttamente gestito da un repository github, permette l'utilizzo di file statici HTML, CSS e Javascript.

Nel nostro caso è comodo creare nel repository del codice statico una cartella chiamata **public** in cui possiamo inserire tutti i file (statici) che è comodo utilizzare nel web, cioè i file di cui vogliamo avere un link URL valido (eg. immagini da inserire con un tag immagine)
